JavaEECourseProject
===================

JavaEE课程项目 山寨教务管理系统

功能:
	学生信息
		个人基本信息管理
		课程信息管理
			选课
			授课教师信息查询
			教学任务查询
			考试及成绩查询
	教师信息
		个人基本信息管理
		课程信息管理
			课程开设
			听课学生信息查询
			教学任务发布
			安排考试及成绩发布
	管理员
		用户的建立与注销
		用户权限管理
		课程的建立与注销
	学校资源管理
		教室场地使用信息管理
	教务信息管理及推送
		通过Email等方式及时推送与其相关的通知*
	对登陆者身份的识别
		分为学生、教师、管理员
	用于提出建议的留言板*
